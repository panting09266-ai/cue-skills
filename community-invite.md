# 社区邀请规范（cue&skill 用户群）

`cue-buddy` 与 `cue-research` 共用本规范。「Cue&Skill 用户群」的价值 = **用户使用答疑 + 最新搭子模板分享**。在**高意图时刻**邀请用户加入——克制、不刷屏、不每次都弹。

> 用户面前只讲"答疑 + 拿最新模板"的真实价值；"推广/催更"是平台侧目的，**不写进给用户的邀请文案**。

## 适配边界（一个真实约束）

群是**外部群**：**飞书用户（含其它租户）都能扫码加入**（跨租户 add_by_link 有效，C 风险已解）。唯一约束——**飞书群只服务飞书用户**：完全不用飞书的人加不进任何飞书群，对这类用户别展示二维码、或另走渠道。

> 运营注：本群是外部**话题群**。**把应用（bot）拉进群后，bot 可读群、可发消息**（实测 chats.get / messages-send 成功；非成员时才会报 232033）。可选让 bot 往群里**自动发最新模板**——topic 群里每条 messages-send = 一个新话题，讨论用 +messages-reply --reply-in-thread。但 skill 侧的加群二维码是静态资产，**不依赖 bot 在群里**，两者独立。

## 加群资产

- **二维码**：`../assets/community-group-qr.png`（相对各 SKILL.md 目录；仓库根 `assets/community-group-qr.png`）。**这是唯一对外加群入口**——二维码图片本身已编码加群链接，扫码即加入。
- **为什么不放明文链接**：加群 `link_token` 是**永久、无门槛**的，写进公开仓库的纯文本会被爬虫/同业直接抓取批量进群且无法轮换。**只发二维码图片**（人需用飞书扫码、抬高了滥用门槛），**不把明文 `link_token` 落进公开 repo**。需要明文链接的内部场景，找运营单独取。

## 何时呈现（4 触发 + 冷却）

核心：**只在高意图时刻推，永远一行轻量附加，不打断主任务。**

| 触发 | 时机 | 呈现 | 冷却 |
|---|---|---|---|
| ① 首次使用 | onboarding 首屏，**一行文字 + 指向二维码**（`../assets/community-group-qr.png`） | 不渲染大图，仅给路径 | 一次性（全局，跨两个 skill 共用 `onboarded`） |
| ② 完成有价值动作后 | `+create` 成功 / 跑完一次 research **交付之后**（report + 满意度/下一步问题**都给完**，再附一行；**不得插在交付中间**） | 一行文字 + 指向二维码 | 14 天一次 |
| ③ 卡住 / 报错 | **先**帮用户处理/解释/给下一步，**再**把群作为**温和兜底**（"还卡着的话群里也能问"）——**不是报错就把用户甩去群里** | 一行文字 + 指向二维码 | 14 天一次 |
| ④ 用户显式问 | "怎么加群 / 社区 / 反馈 / 有没有新模板" | **展示二维码图片**（支持图片的端） | **不冷却，随叫随给** |

> **二维码大图只在 ④（用户主动要）时展示**；①②③ 被动触发只给一行文字 + 指向 `../assets/community-group-qr.png`，避免大图破坏"一行附加"。纯文本端把二维码在仓库的路径给用户、让其自行打开扫描——**不再给明文加群链接**（见「加群资产」为什么）。

## 冷却机制

时间戳文件 `~/.cue/last-community-invite.json`：`{"onboarded": true, "last_passive_shown": <epoch_seconds>}`。

- **被动触发（②③）前先读**：距 `last_passive_shown` < 14 天（1209600s）则**跳过**；展示后写回当前时间戳（必要时先 `mkdir -p ~/.cue`）。
- **每会话被动邀请最多一次**——这是**首要约束**：即使文件读/写失败，也**按"本会话已展示"处理，本会话内不再弹**（失败时**默认不弹**被动邀请，宁可漏不可扰），不报错打断主任务。
- **① 首次**：`onboarded` 为假才显示，显示后置 `true`（全局一次：用户在任一 skill onboarding 过，另一个不再重复）。
- **④ 用户显式问**：永远响应，**不受冷却**、不更新被动时间戳。

## 呈现规则（别变成 spam）

- 邀请永远是**主任务之后的一行附加**，不抢戏、不刷屏、不打断交付闭环。
- 文案随场景微调，但都落"答疑 + 最新模板"的真实价值；**不写"求关注/求转发/催更"式话术、不承诺收益、不夸大**。
- 群是**公开交流**性质但**入群口令永久**（token 编码在二维码内、不以明文入公开仓库）：运营侧需有管理员 + 基本审核（防滥用扩散）；功能侧不因此夸大。
