# 示例：季度财报分析点评

> 完整可参考的种子模板：金融 / 投研业务 / 财报分析场景。已通过 `+validate` 校验，可直接 `+create` 或在此基础上 `+author` 拷贝裁剪。

业务背景：二级市场分析师或投资者在上市公司发布财报后，需要快速拆解核心财务指标、增长归因、盈利能力变化以及管理层前瞻指引，以判断业绩是否超预期并评估后续投资风险。

### `search_plan`

```markdown
**[[业绩概览] 最新季报与核心财务]**
- **数据路由**: A 股 / 港股最新季报、中报、年报 / 公告披露的业绩快报 / 核心财务指标 API
- **执行动作**: 拉取本季度营收、归母净利润、扣非净利润、毛利率、净利率；计算同比、环比增幅；核对分业务板块的营收构成与毛利贡献
- **验证策略**: 优先采纳审计后的年报数据；季报未经审计时需标注"未经审计"；计算差异 >1% 时以官方公告披露的增幅为准

**[[经营解读] 业绩说明会与管理层讨论]**
- **数据路由**: 业绩说明会纪要 / 投研路演笔记 / 管理层讨论与分析 (MD&A) / 交易所互动平台回复
- **执行动作**: 提取管理层对本季度增长驱动的定性描述、产能利用率、订单排期、原材料成本压力及未来指引
- **验证策略**: 管理层指引与实际数据冲突时，标注"指引执行偏离"；多份纪要口径不一时，以公司官方发布的正式版本为准

**[[同业对标] 行业景气与竞争格局]**
- **数据路由**: 行业研报 / 同业可比公司财报 / 行业协会统计数据 / 价格监测数据
- **执行动作**: 选取 2-3 家同业代表性公司，对比其同期增速、利润率与存货周转率；获取所属行业本季度的整体景气水平（如销量增速、平均售价 ASP 变动）
- **验证策略**: 同业对比需注明可比公司名单；行业数据来源需具备公信力（如专业咨询机构或官方统计）

**信源偏好**: 官方财报 > 业绩说明会官方纪要 > 交易所互动平台 > 行业协会 > 权威研报 > 媒体报道
**避坑指南**:
- 注意区分"归母净利润"与"扣非归母净利润"，后者更能反映主业经营真实情况
- 财报金额单位（万元/亿元）必须准确核对，防止数量级错误
- 港股与 A 股财报准则差异（如折旧处理）需在对比时标注
- 业绩说明会纪要中的"口头预测"不应等同于"确定性结果"

**数据裁决逻辑**:
1. 财务指标以最近一期经审计年报或官方披露季报为准
2. 营收/利润增速计算与官方通稿不一致时，以官方通稿为准
3. 多源纪要冲突时，优先采纳公司官网发布的正式版本
4. 同业对标数据缺失时，使用行业平均值兜底并明示假设
```

### `report_format`

```markdown

> - **报告类型**: 季度财报分析点评
> - **基调设定**: 客观中立
> - **核心命题**: 拆解季度业绩波动的底层逻辑，为投资者提供高效决策参考

# [目标_财报_主体] 季度财报分析点评
报告时间: YYYY年MM月DD日

## 1. 核心观点与业绩定调

> **[执行蓝图]**
> - **研究目标**: 定调本季度业绩是超预期、符合预期还是低预期
> - **逻辑链条**: 核心指标对标预期 → 增长/下滑主因 → 未来趋势预判
> - **信息需求**: 营收/净利增长率、市场共识预期数据、管理层指引
> - **输出形式**: 结论先行的三段式点评

## 2. 核心财务指标概览

> **[执行蓝图]**
> - **研究目标**: 展示营收与利润的规模及增速
> - **逻辑链条**: 营收同比/环比 → 归母净利润同比/环比 → 扣非净利润表现
> - **信息需求**: 最新三期利润表核心数据
> - **输出形式**: 关键指标对比表 + 趋势说明

## 3. 盈利能力拆解

> **[执行蓝图]**
> - **研究目标**: 分析毛利率与净利率波动的核心驱动力
> - **逻辑链条**: 毛利率变动归因（量价/成本/结构） → 期间费用率变化 → 最终净利率
> - **信息需求**: 分产品/分地区毛利率、销售/管理/研发/财务费用明细
> - **输出形式**: 盈利能力变动分析段落

## 4. 营运质量与现金流

> **[执行蓝图]**
> - **研究目标**: 评估业绩含金量与潜在经营风险
> - **逻辑链条**: 存货/应收账款周转 → 经营性现金流净额 vs 净利润比率
> - **信息需求**: 资产负债表周转项、现金流量表经营项
> - **输出形式**: 营运质量评估段落

## 5. 管理层前瞻与行业指引

> **[执行蓝图]**
> - **研究目标**: 捕捉后续经营动向与行业景气拐点
> - **逻辑链条**: 业绩会要点提炼 → 行业销量/价格趋势 → 产能与订单储备
> - **信息需求**: 业绩说明会纪要、行业研报、管理层前瞻陈述
> - **输出形式**: 前瞻性观点汇总清单

## 6. 自查与重点关注

> **[执行蓝图]**
> - **研究目标**: 提示本次财报中需要深度挖掘的疑点或后续风险点
> - **逻辑链条**: 异常指标识别 → 潜在风险归因 → 建议关注事项
> - **信息需求**: 报表异常项、商誉、大额减值、诉讼、行业政策变化
> - **输出形式**: 风险提示清单

## 7. 免责声明

> **[执行蓝图]**
> - **研究目标**: 声明信息来源与合规边界
> - **逻辑链条**: 数据来源说明 → 非投资建议声明
> - **信息需求**: 官方披露口径、免责声明标准话术
> - **输出形式**: 免责段落

---

> 本报告基于公开财报及官方披露编制，不构成任何投资买卖建议。
> 数据时效以官方公告日为准。
```
